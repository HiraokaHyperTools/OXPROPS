| Data type | PtypString, 0x001F |
| Area | Email |
| Defining reference | [MS-OXOMSG] section 2.2.2.34 |
| Consuming references | [MS-OXCMAIL] |
| Alternate names | PR_DSN_REMOTE_MTA, ptagDsnRemoteMta |
