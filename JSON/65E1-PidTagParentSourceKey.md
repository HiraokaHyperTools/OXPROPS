# PidTagParentSourceKey

| Key | Value |
|---|---|
| Canonical name | PidTagParentSourceKey |
| Description | Contains a value on a folder that contains the PidTagSourceKey property (section 2.1016) of the parent folder. |
| Property ID | 0x65E1 |
| Data type | PtypBinary, 0x0102 |
| Area | ExchangeNonTransmittableReserved |
| Defining reference | [MS-OXCFXICS] section 2.2.1.2.6 |
| Alternate names | PR_PARENT_SOURCE_KEY |
